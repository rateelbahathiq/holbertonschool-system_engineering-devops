Web Infrastructure Design

This project is about designing different web infrastructures.

We start with a simple setup using one server, then move to a distributed system, then a secured and monitored setup, and finally a scalable architecture.

The goal is to understand how real systems work and how they improve step by step.

Files:

0-simple_web_stack
Basic one server setup.

1-distributed_web_infrastructure
Two servers with a load balancer.

2-secured_and_monitored_web_infrastructure
Adds security (HTTPS, firewalls) and monitoring.

3-scale_up
More advanced setup with separated components and better scalability.
